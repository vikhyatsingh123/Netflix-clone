# Netflix Clone

## Technologies used

### Front end
- HTML / CSS / JS / TypeScript
- ReactJS

### deployment into production

- Front end web: github
- Api: themoviedb

## How to run the project


### Front end web
prerequisites: npm / yarn

```bash
# clone repository
git clone

# enter the front end web project folder
cd front-end

# install dependencies
npm install

# run the project
npm start
```
